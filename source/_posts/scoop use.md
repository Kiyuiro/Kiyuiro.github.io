搜索包含 `name` 的程序
```
scoop search [name]
```

安装 `name` 程序, 名称必须与仓库中的完全一致
```
scoop search [name]
```

查看添加的仓库源
```
scoop bucket list
```

添加新的仓库源地址
```
scoop bucket add [soure name]
```
